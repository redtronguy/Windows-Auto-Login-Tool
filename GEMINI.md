# GEMINI.md

## Project Overview

This project is a PowerShell script that provides a text-based UI for managing Windows Auto-Login. The script allows users to enable or disable auto-login, and it includes logging and error handling.

## Building and Running

The script can be executed in a PowerShell environment. No build steps are required.

To run the script, open a PowerShell terminal, navigate to the project directory, and execute the following command:

```powershell
.\Windows-Auto-Login-Tool.ps1
```

## Development Conventions

All actions are logged to a `logs` directory in the project's root.

*   `Windows-Auto-Login-Tool.ps1`: The main script for enabling and disabling auto-login.
